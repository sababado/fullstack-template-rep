# [Project Name] Template Repository

This repository serves as a starter kit for new projects. It includes a standard directory structure and a comprehensive suite of documentation templates to ensure consistency across the organization.

## Contents

- **template-project/**: The core template structure to copy for new projects.
    - **apps/**: Frontend and Backend applications.
    - **packages/**: Shared packages (UI Kit).
    - **docs/**: Project-wide documentation.

## How to Use

1. **Clone this repository.**
2. **Copy the `template-project` folder** to your new project location.
3. **Rename the folder** to your project name.
4. **Search and Replace:**
    - Search for `[Project Name]` and replace with your actual project name.
    - Search for `[Repo URL]` and replace with your repository URL.
5. **Initialize Git:**
    ```bash
    cd your-new-project
    git init
    ```
6. **Follow the Setup Instructions** in `docs/README.md`.

## Maintaining this Template

- **Contributing:** Updates to the template should be made via Pull Requests.
- **Versioning:** This template follows Semantic Versioning. See `VERSIONING.md` for details (in the template docs).
